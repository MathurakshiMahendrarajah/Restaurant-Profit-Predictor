# Restaurant Profit Predictor

Predict potential profits for a new restaurant based on city population using Linear Regression.

## Project Overview

This project is a Machine Learning web application that predicts the expected profit for a new restaurant in a given city based on the city’s population. It uses linear regression and a trained model to make predictions and is deployed as a Flask web app with a modern, responsive frontend.
	
  •	ML Model: Linear Regression trained on historical data (population vs profit) 
	
  •	Frontend: Bootstrap 5 for responsive design, carousel for restaurant images
	
  •	Backend: Flask API serving predictions (/predict)
	
  •	Deployment: Live on Render.com

## Features

  •	Responsive web interface
	
  •	Carousel showcasing restaurant images
	
  •	Input for city population with up/down controls
	
  •	Automatic scaling of input values for model compatibility
	
  •	Predicted profit displayed dynamically
